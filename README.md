# 🎮 Void Runner

A browser game built with Flask and vanilla JavaScript.  
Click the glowing dot, get sucked into the white void, and survive the shadows hunting you.

## 🚀 How to Run

1. Clone the repo:
   git clone https://github.com/armen-marg/My_Game.git
   cd My_Game

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   python app.py

4. Open your browser at:
   http://127.0.0.1:5000

## 🕹️ How to Play

- Move your mouse to control the black dot
- Collect orbs to score points
- Avoid the shadow hunters chasing you
- You have 3 lives — don't get caught!
- Game gets faster over time

## 🛠️ Built With

- Python + Flask (backend)
- HTML / CSS / JavaScript (game engine)
- Canvas API (rendering)
